#spesifikasi
=> project ini adalah dari tema yang kedua yaitu
   TO DO LIST
=> jadi program ini kita bisa membuat sebuah
   catatan baru, kita bisa menambahkannya,
   kita juga bisa mengedit catatan tersebut dan
   kita juga bisa menghapus catatan tersebut.
=> jadi sebenarnya jika kita menjalankan
   program ini kita tidak akan kesulitan lagi
   untuk khawatir salah sebab disini kita
   dikasih tau langkah-langkah untuk menginput
   dan program ini mempermudah kita untuk mengingat
   catatan-catatan kita.
